## Flight Price Tracker

A full stack web application that tracks and visualizes flight prices to help users find the best time to buy tickets

### Features
- Live flight search using external APIs
- Custom filtering by airline and sorting by price or duration
- Dedicated dashboard to view saved flights
- Interactive pricing graph to visualize trends
- Secure backend connection to a local MySQL database

### Tech Stack

- Frontend: React
- Backend: Python and Flask
- Database: MySQL
- Security: Environment variables for credential management
